## Issue's
Ben jij een bug tegen gekomen of crashde de app op jou telefoon?

    Titel: [Crash] wat informatie
    Omschrijving:
    Toestelnaam: GT-3254i
    Android versie: 5.0.1
    Omschrijving: blablabla
